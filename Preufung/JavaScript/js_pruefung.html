Die Aufgaben

Aufgabe 1: Wenn den button (Durchschnitt) geklick wird, wird der Durchschnitt von Note1 bis Note4 kalkuliert und im Element mit id #note geschrieben.
Es muss erst gecheckt ob Note1, Note2, Note3 ODER Note4 NICHT kleiner 0 oder größer 100 sind. Es wird den Fehler in der id #fehler eingeblendet (display block) sondern ausgeblendet (display none).

Aufgabe 2: Wenn den button (Bewertung) geklick wird, wird den Durchschnitt bewertet.
Die Bewertung wird im #bewertung Element geschrieben. ("Befriedigend", "Gut", "Sehr gut")
nach den folgenden Werten:

Durchschnitt Bewertung
gleich oder kleiner 80 Durchschnitt <= 80 => "Befriedigend"
größer 80 und gleich oder kleiner 91 Durchschnitt > 80 und Durchschnitt <=  91 => "Gut"
größer 91 Durchschnitt > 91 => "Sehr gut"
